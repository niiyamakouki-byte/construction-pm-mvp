> **対象エリア**: 東京都23区  
> **価格基準**: 2025-2026年 都内相場

> **単価区分**: 材工共（材料費+施工費）を基本とする

## 価格変動要因

| 要因 | 影響 |
|------|------|
| 高層階（EV無し3F以上） | +10~30% |
| 搬入困難（狭小路地・階段のみ） | +15~30% |
| 夜間作業 | +25~35% |
| 休日作業 | +25~50% |
| 小規模工事（50万円以下） | 諸経費率UP |
| 繁忙期（3月・9月） | +5~15% |
| マンション共用部制約 | 養生費UP |
